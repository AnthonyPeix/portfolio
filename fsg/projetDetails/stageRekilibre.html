Stage chez Rékilibre

PHP - Symfony - Bootstrap

Durant mon stage de première année, j'ai pu participer à l'amélioration d'un site web tenu par l'association Rékilibre. Ce site internet avait pour objectif de gérer les différents évènements de l'association (activités, réunion, etc...) et permettait aux futurs adhérents de pouvour s'y inscrire voir s'y réinscrire et dans ce cas, pouvoir payer leur frais d'adhésion.

Mes différentes missions :


  • Correction de bug : le site internet avait quelques bugs majeurs comme une impossibilité pour les anciens membres de pouvoir se réinscrire entre autre. L'objectif était donc de corriger ces différents problèmes.

  • Refonte du formulaire d'inscription et API bancaire : A la base, chaque adhérent devaient s'inscrire d'une part et envoyer un chèque physiquement à l'association, procédure fastidieuse pour eux et surtout pour les organisateurs. Nous sommes donc convenu de pouvoir payer les frais d'inscription directement sur le formulaire d'inscription de l'association. J'ai donc remanié le formulaire et a du réfléchir avec l'équipe sur comment on allait s'y prendre on fonction de nos moyens. Etant association, celle-ci avait un compte bancaire spécialisé pour ce status chez une banque. Celle-ci proposait une API permettant d'intégrer facilement une méthode de paiment sur notre site internet. Nous sommes parties sur cette voie-ci.

  • Migration de version du site internet : La fin de mon stage, bien que non finalisé sur cette tâche était ciblé sur la possiblité de migrer notre site internet. Celui-ci était sur une ancienne version du Framework de Symfony, une version qui aujourd'hui, est obsolète. J'ai donc fais pas mal de test de recherche pour passer à une version bien plus récente de Symfony. Cette tâche ayant prit beaucoup de temps, je n'ai pas pu l'aboutir en juste quelques journées.
