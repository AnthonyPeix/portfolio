Projet Java – INFOWARE-SIRH

Java - MySQL

Du 15 au 19 avril 2024, nous avons réalisé un projet ambitieux visant à développer une application de gestion d’entreprise en Java. Ce projet avait pour objectif de créer une interface utilisateur intuitive ainsi que des fonctionnalités essentielles de gestion de données.

Nous avons également utilisé SQL avec PhpMyAdmin pour visualiser, manipuler et structurer les données dans une base MariaDB.

Le projet a été confié à STESIO par INFOWARE, un distributeur de matériel informatique. Il s’agissait de moderniser leur système de gestion des ressources humaines via le développement d’un nouveau logiciel baptisé INFOWARE-SIRH.

Nous avons débuté par l’implémentation d’un module CRUD (Create, Read, Update, Delete) permettant de gérer les données des salariés. Cette fonctionnalité constitue le socle d’un système RH plus complet et performant.

Ce projet vise à améliorer l’efficacité interne, la fiabilité des données RH, ainsi que l’expérience utilisateur pour les employés et les gestionnaires.
