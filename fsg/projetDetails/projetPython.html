Projet Python – Script de gestion d’utilisateurs

Python - MySQL

Ce projet a été réalisé lors de ma première année de BTS SIO. L’objectif était de développer un script Python capable de gérer des utilisateurs, de manière à automatiser la création de fichiers SQL pouvant ensuite être injectés dans une base de données.

Ce script offre une solution simple et efficace pour générer rapidement les requêtes nécessaires à l’ajout d’utilisateurs dans une base MySQL, améliorant ainsi la rapidité d’intégration et la fiabilité des données insérées.

Ce projet m’a permis de renforcer mes compétences en scripting Python, en gestion de bases de données, ainsi qu’en génération de fichiers dynamiques en fonction des entrées utilisateur.
